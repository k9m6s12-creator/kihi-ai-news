# 毎朝のAIニュース・ダイジェスト（Webページ）

このフォルダ（`web/`）だけで、ひとつの「ホームページ」として成立します。
中身は3つだけ。

- `index.html` … 見た目とページの仕組み（これをブラウザで開く）
- `digests.js` … 中身のデータ（毎日のダイジェストがここに溜まる）
- `README.md` … この説明書き

## いま、すぐ見る方法（公開しなくてもOK）
`index.html` を**ダブルクリック**すると、自分のPCのブラウザで表示できます。
（インターネットには出ていない＝自分だけが見られる状態）

## スマホでどこでも見たいときは「公開（ホスティング）」する
このフォルダをネット上の置き場に上げると、URLが発行され、スマホからも見られます。
やり方は `reference/Web公開ガイド_やさしい解説.md` を参照。

## 毎日の更新の仕組み
`/news-digest` スキルが、新しい1日分を `digests.js` の先頭に追記します。
`index.html` は触りません（データだけ増える設計）。
